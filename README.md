# Simple C++ Calculator

# Overview
A build a basic calculator program written in C++ that performs simple arithmetic operations between two numbers.

# Features
Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)

# How to Use
Enter the first number
Enter an arithmetic operator (+, -, *, /)
Enter the second number
The program displays the result

# Requirements
C++ compiler (g++, CodeBlocks, Dev-C++, etc.)
